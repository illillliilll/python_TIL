## 01) 정수 n을 입력받고 1부터 n까지의 수 중에서 소수의 개수 출력
```python
n = int(input()) # 1
count = 0 # 2

for i in range (2, n + 1): # 3
    is_prime = True

    for j in range (2, i): # 4
        if i % j == 0:
            is_prime = False # 5
            break

    if is_prime: # 6
        count += 1 # count = count + 1

print (count) # 7
```

➜ 정수 n 입력 받기

#### #2

#### #3

#### #4
➜ i가 소수인지 확인하고 2부터 i-까지 나눠봄 // 나누어 떨어지면

#### #5

#### #6
➜ 소수라면 count 증가

➜ count가 소수의 개수를 저장한 변수이므로 count 출력
